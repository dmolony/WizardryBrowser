# Wizardry Browser
This code requires the DiskBrowser.jar file to be on the classpath.
### MacOS/Linux Execution
java -classpath /path/to/WizardryBrowser.jar:/path/to/DiskBrowser.jar com.bytezone.wizardry.ScenarioBrowser
### Windows Execution
java -classpath /path/to/WizardryBrowser.jar;/path/to/DiskBrowser.jar com.bytezone.wizardry.ScenarioBrowser
